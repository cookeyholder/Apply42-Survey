# 實作任務清單

## CSS 基礎建設
- [ ] 定義 CSS 變數 (`:root`) 設定新配色（Primary, Secondary, Background, Text）
- [ ] 引入現代字型（如 Google Fonts: 'Noto Sans TC' 或 'Inter'）

## 元件樣式更新
- [ ] **卡片 (Cards)**: 更新陰影樣式、圓角、移除舊有左側粗邊框設計，改用更現代的頂部強調或整體極簡風。
- [ ] **按鈕 (Buttons)**: 更新按鈕顏色、圓角、Hover/Active 效果。
- [ ] **表單 (Forms)**: 更新 Input/Select 的 Focus 狀態樣式。

## 互動效果
- [ ] 為可互動元素添加 `transition` 屬性。
- [ ] 實作卡片 Hover 浮起效果。

## 驗證
- [ ] 檢查所有頁面元素是否符合新配色。
- [ ] 確認手機版（RWD）顯示正常。
- [ ] 確認無紫色殘留。
