## ADDED Requirements

### Requirement: 統計前端共用模組初始化
系統 MUST 提供可在學生端與老師端重複使用的統計前端共用模組，並以單一初始化呼叫完成事件綁定與初始載入。

#### Scenario: 在學生端初始化成功
- **WHEN** 學生端頁面載入並執行統計模組初始化
- **THEN** 統計頁籤首次顯示時會正確載入摘要資料且不拋出 ReferenceError

#### Scenario: 在老師端初始化成功
- **WHEN** 老師端頁面載入並執行統計模組初始化
- **THEN** 統計頁籤首次顯示時會正確載入摘要資料且與學生端行為一致

### Requirement: 腳本注入位置安全
系統 SHALL 只在 HTML 的 `<script>` 區塊注入統計共用模組，且不得將腳本內容放入 `<style>` 或其他非腳本區域。

#### Scenario: 模板渲染時腳本位置正確
- **WHEN** 頁面由 HtmlService 渲染
- **THEN** 共用統計腳本可被瀏覽器視為有效 JavaScript 執行，且不造成整頁 CSS/HTML 解析錯誤

### Requirement: 事件綁定去重與可重入保護
系統 MUST 避免統計 tab 事件監聽重複綁定，並確保多次切換頁籤不會造成重複請求。

#### Scenario: 多次切換頁籤
- **WHEN** 使用者多次切換到統計頁籤
- **THEN** 系統只會在首次需要時執行初始載入，後續不重複綁定同一監聽器

### Requirement: 回退相容
系統 SHALL 保留回退到雙檔內嵌統計腳本的可行路徑，以便共用模組異常時快速恢復服務。

#### Scenario: 共用模組導入後發生回歸
- **WHEN** 上線後發現統計頁無法正常載入
- **THEN** 維運可在單一回退操作中恢復到先前可用版本並讓統計頁恢復服務
