# 階層式學校-科系選擇

## 摘要
將志願選擇從單一下拉選單改為先選學校、再選科系的兩階段選擇。

## 動機
目前學生需要在一個包含所有校系的長列表中選擇志願，使用體驗不佳。改為先選學校再選科系可以：
- 減少選項數量，更容易找到目標
- 避免選錯學校的錯誤
- 提供更清晰的操作流程

## 範圍
- 僅影響 `index.html` 中的志願選擇 UI
- 不影響後端資料結構和表單提交

## 驗收標準
- [ ] 每個志願有兩個下拉選單：學校和科系
- [ ] 選擇學校後，科系選單動態更新
- [ ] 保持原有的重複檢查和學校志願數限制
- [ ] 已選擇的志願正確回顯
